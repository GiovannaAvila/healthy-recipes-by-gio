# Healthy Recipes by Gio

Site estático para reunir receitas saudáveis, links de redes sociais e formulário de interesse para comunidade.

## Estrutura

- `index.html` — página inicial do site
- `assets/styles.css` — estilos visuais
- `assets/script.js` — menu mobile e interação do formulário
- `recipes/` — páginas individuais das receitas
- `assets/images/` — imagens das receitas

## Próximos ajustes

1. Trocar os links das redes sociais pelos links reais da Gio.
2. Substituir as receitas de exemplo pelas receitas finais.
3. Adicionar fotos reais das receitas.
4. Conectar o formulário de email a uma ferramenta real como Mailchimp, ConvertKit, Formspree ou Google Forms.
5. Ativar o GitHub Pages nas configurações do repositório.
